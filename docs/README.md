### 3 年後になりたい姿

- フロントエンドの専門性を高め、他の人に負けない設計スキルを身につける
- 定期的に OSS にコントリビュートする

### 5 年後になりたい姿

- リードポジションとしてプロジェクトを牽引し、変更の容易なアプリケーションを実現する

- フロントエンドと設計に関するスキルが足りていない

### なりたい姿を達成するために、取り組みたいこと

- typo, 翻訳などできそうなことを見つけて、まずは 1 回 OSS にコントリビュートする
