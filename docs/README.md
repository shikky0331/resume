## 将来どうなりたいか

### 3 年後になりたい姿

- フロントエンドの専門性を高め、他の人に負けない強みを獲得する
- 定期的に OSS にコントリビュートする
- 技術の話を英語でできるようになる
- 技術イベントへの登壇・技術記事のアウトプットを定期的におこなう

### 5 年後になりたい姿

- リードポジションとしてプロジェクトを牽引する
- OSS のメンテナーになる
- ビジネスで英語を問題なく使用する
- フロントエンドの界隈で認知される

### 現在となりたい姿とのギャップ

- 自分の強みを把握できていない。また、高めていきたい技術領域がフロントエンドと範囲が絞れていない
- OSS にコントリビュートしたことがない
- 簡単な英語しか話すことができない
- 外部に公開するアウトプットの量が少ない

### なりたい姿を達成するために、取り組みたいこと

- プライベート学習の中で自分の強みを見つけ、高めていきたい技術領域を絞る
- typo, 翻訳などできそうなことを見つけて、まずは 1 回 OSS にコントリビュートする
- 毎日の英語学習を継続する
- 月 1 ペースで外部に公開できるコンテンツをアウトプットする
